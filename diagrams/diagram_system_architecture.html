QMIX System Architecture — CTDE Pipeline
Environment
Simulator
Overcooked-AI
3 layouts · gym
horizon = 400 steps
2 chefs / episode
Cooperative kitchen simulator. Two agents collaborate to prepare and deliver onion soups. Three layouts of increasing difficulty: cramped_room, coordination_ring, counter_circuit. Episode truncates at 400 steps. Agent indices are randomized at each reset.
Team Reward
r_team
+20 delivery
shaping: pot/dish/soup
values: +3/+3/+5
r_team = r_env + shaped_r₀ + shaped_r₁. The shared team reward is used for the Bellman target. Shaped rewards are given per-agent by the environment for specific events (potting an onion, picking up a dish, picking up soup).
Observations
Agent 0 Obs
s₀ ∈ ℝ⁹⁶
orientation · distances
held item · pot states
partner details
96-dim state. Includes: own orientation, distances to dispensers and serving area, held item, cooking pot distances and timers, and the same details for the partner agent. Fully observable environment — "observation" = full state.
Agent 1 Obs
s₁ ∈ ℝ⁹⁶
orientation · distances
held item · pot states
partner details
Same structure as s₀ from agent 1's perspective. Agent index randomization forces both agents to learn identical role-agnostic policies — validated empirically through behavioral metric analysis.
Centralized Info
z = [s₀; s₁] ∈ ℝ¹⁹²
training only
→ hypernetworks
Concatenation of both agents' states. Used only during training as input to the hypernetworks that generate mixing network weights. Not available to agents during execution — this is what makes it CTDE.
Agent Networks
Agent 0 DQN θ₀
Q₀(s₀, a ; θ₀)
96→256→256→6
ReLU · Q-values
+ target Q̄₀
Per-agent deep Q-network. Outputs Q-values for all 6 actions given s₀. Two hidden layers of 256 units each with ReLU. Separate target DQN Q̄₀ for stable Bellman targets, soft-updated with τ=0.028.
Agent 1 DQN θ₁
Q₁(s₁, a ; θ₁)
96→256→256→6
ReLU · Q-values
+ target Q̄₁
Identical architecture to Q₀ but separate parameters. Agents learn independently. During ε-greedy training, each makes its own random/greedy decision.
Replay Buffer D
100K transitions
(s₀,s₁,z,a₀,a₁,
r_team,d,s₀',s₁',z')
batch = 512
Off-policy experience replay. Stores full transition tuples. Uniform random sampling breaks temporal correlations. Pre-filled with random-policy transitions before training begins.
Mixing (Training)
HyperNets W₁,W₂,B₁,B₂
z → weights
Linear(192→128)→Softplus
W₁:(B,64,2) W₂:(B,1,64)
biases unconstrained
Four hypernetworks (single Linear layers each) take centralized info z and produce the weights and biases for the mixing network. Softplus on W₁,W₂ ensures strictly positive weights — the mathematical requirement for the IGM monotonicity guarantee.
Mixing Network · CORE
fmix → Q_tot
ELU(W₁·[Q₀;Q₁]+B₁)
→ W₂·h + B₂
→ scalar Q_tot
Takes [Q₀, Q₁] as a 2-vector. Applies two linear layers using hypernetwork-generated weights. ELU activation in hidden layer. Softplus weights guarantee strict monotonicity: ∂Q_tot/∂Qᵢ > 0 for all i. This ensures local greedy actions are globally optimal.
Target Mixer
Q_tot_next
soft-updated copy
τ = 0.028 / epoch
Target mixing network used to compute Q_tot for next states in the Bellman target. Prevents moving-target instability. Updated alongside Q̄₀ and Q̄₁ at end of each training epoch.
Optimization
Bellman Target
targetQMIX
r_team + γ·Q_tot_next
·(1−done) · γ=0.99
One-step TD target using team reward and discounted next-state Q_tot from target networks. γ=0.99 chosen after sweep — γ=0.999 caused agents to overlook immediate sub-tasks.
Loss · AdamW
Smooth L1
Huber(Q_tot, target)
lr = 0.0007
batch = 512
Smooth L1 (Huber) loss is more robust to outlier Q-value estimates than MSE during early high-variance exploration. AdamW lr=0.0007 chosen after sweep — lower delays convergence, higher causes Q-value overestimation and collapse to zero reward.
Decentralized Execution
Agent 0 Acts
argmax Q₀(s₀)
s₀ only · no comms
no centralized info
fully independent
At execution time, agent 0 selects actions by argmax over Q₀ given only its local observation s₀. No access to s₁ or the mixer. The IGM property guarantees this is globally optimal.
Agent 1 Acts
argmax Q₁(s₁)
s₁ only · no comms
no centralized info
fully independent
Agent 1 acts symmetrically and independently. Both agents converge to near-identical policies due to agent-index randomization, validated through behavioral metric analysis showing ~50/50 task distribution.
ε-Greedy (training)
Exploration
ε·exp(-λ·epoch)
λ=0.00055 · ε_min=0.005
Exponential ε-decay during training. λ=0.00055 is the most sensitive hyperparameter — the margin between successful convergence and failure across all three layouts is extremely narrow.
