VDN's approach
VDN — Linear Decomposition (Predecessor)
Mixing Function
Q_tot = Q₀ + Q₁
A simple arithmetic sum. No network. No parameters. No conditioning on global state. The team value is just the linear sum of individual utilities.
IGM — How it's satisfied
Trivially — by linearity
Linear sum is trivially monotonic in every Qᵢ. IGM holds automatically without any architectural constraint. Simple, but this simplicity comes at a cost.
Limitation
Assumes additive independence
Agents' contributions are assumed to be independent and additive. In Overcooked, an agent holding a soup dish near the serving area makes its partner's potting action far more valuable — a non-linear interaction that Q₀ + Q₁ cannot express.
Limitation
No global state conditioning
Q_tot has no access to the joint state. The mixing weights are fixed — they cannot shift based on context (where both agents are, what both hold).

QMIX's extension
QMIX — Monotonic Nonlinear Mixing (Extension)
Mixing Function · CORE CHANGE
Q_tot = fmix(Q₀, Q₁ ; z, θ_mix)
Replaces the fixed linear sum with a learned nonlinear function conditioned on the full joint state z. Can express any monotone relationship between individual utilities and team value.
New: Hypernetworks
z → mixing weights dynamically
Four single-layer hypernetworks take z = [s₀; s₁] as input and produce the weights and biases of the mixing network. The mixing function adapts to context at every timestep.
IGM — How it's satisfied
Architecturally enforced via Softplus
Softplus activation on mixing weights guarantees ∂Q_tot/∂Qᵢ > 0 for all i — strict monotonicity is built into the network, not assumed. More principled than VDN's trivial linearity.
What QMIX adds over VDN
Nonlinear mixing function fmix — captures inter-agent contribution effects
Centralized state z = [s₀; s₁] conditions the mixing — global context matters
Hypernetworks generate dynamic weights — contribution weights shift with state
Strict Softplus monotonicity — IGM is enforced, not assumed by linearity
⚡ IGM Property
argmax_a Q_tot ⟺ [argmax_{aᵢ} Qᵢ(sᵢ, aᵢ)]
Both VDN and QMIX satisfy IGM — but QMIX satisfies it for a much richer class of value functions. VDN's satisfaction is a trivial consequence of linearity. QMIX's is an explicit architectural guarantee that scales to non-linear cooperative interactions.
