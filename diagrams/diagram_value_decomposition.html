Value Decomposition — VDN vs QMIX
Predecessor
VDN
Value Decomposition Networks
Q_tot = Σ Qᵢ(sᵢ, aᵢ ; θᵢ)Linear sum — assumes equal, additive contribution per agent
How It Works
Linear Factorization
The global team value function is simply the sum of individual agent Q-values. No mixing network. No conditioning on global state. Each agent's contribution is treated as independent and additive.
Advantage
Simplicity
No additional parameters. Linear sum trivially satisfies IGM — greedy local actions are always globally consistent.
Limitation
Weak Representational Power
Agents' contributions to team value are almost never truly linear in real cooperative tasks. Collaboration, spatial dependencies, and role asymmetry create non-linear interaction effects that VDN cannot capture.
Limitation
No Global Context
The team Q-value has no access to the joint state. Coordination-critical global information (where both agents are, what both are holding) cannot influence the decomposed values.
Implemented
QMIX
Monotonic Value Function Factorisation
Q_tot = fmix(Q₁,…,Qₙ ; z, θ_mix)Nonlinear mix conditioned on global state z — via hypernetworks
How It Works · CORE
Monotonic Nonlinear Mixing
The mixing function fmix is a 2-layer neural network whose weights are generated by hypernetworks conditioned on the joint state z. This allows arbitrary non-linear combinations of agent Q-values — capturing complex inter-agent dependencies — while the Softplus weight constraint enforces strict monotonicity.
Advantage
Rich Factorization
Can represent any monotone function of individual Q-values. State-conditioned weights let the mixer assign context-dependent contribution weights to each agent.
Advantage
IGM Guaranteed by Design
Softplus activation on mixing weights ensures ∂Q_tot/∂Qᵢ > 0 for all i — strict monotonicity is built into the architecture, not assumed.
⚡ IGM Property — The Mathematical Core
argmax_a Q_tot(s, a) ⟺ [argmax_{aᵢ} Qᵢ(sᵢ, aᵢ)]
The Individual-Global-Max property: greedy local actions compose into the globally optimal joint action. This is what makes decentralized execution work without communication. Monotonicity is a sufficient condition: if Q_tot is strictly increasing in every Qᵢ, then maximizing each Qᵢ independently always maximizes Q_tot jointly.
